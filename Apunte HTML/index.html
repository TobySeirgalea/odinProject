Etiquetas sobre semántica de texto:

Entre las etiquetas que nos permiten dar semántica al texto tenemos:

  • em: Sirve para poner énfasis, agregando un estilo itálico. Funciona para screen readers
  • strong: Sirva para indicar importancia, agrega un estilo bold o negrita al texto. También funciona para screen readers.
